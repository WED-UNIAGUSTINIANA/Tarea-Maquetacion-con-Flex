Ejercicios para la tarea:

  1. Cambia la dirección del contenedor principal para que el sidebar esté a la derecha en lugar de a la izquierda.
  2. Modifica la galería para que los elementos se muestren en 4 columnas en lugar de 3.
  3. Ajusta el flex-grow y flex-shrink de las tarjetas para que se expandan de manera diferente.
  4. Cambia la alineación de los elementos en el header para que estén a la izquierda en lugar de centrados.
  5. Haz que el footer se convierta en una columna en dispositivos móviles en lugar de una fila.
  6. Implementa un menú horizontal en el sidebar usando flex en lugar de la columna actual.
  7. Crea un nuevo componente de perfil de usuario usando flex para alinear la imagen y el texto.

Desafío adicional:

Crea un sistema de rating con estrellas usando flex. Debe incluir 5 estrellas que se pueden colorear de forma parcial para mostrar puntuaciones como 4.5 estrellas.
